# E-Commerce Backend

## Description

This is an application that mainly focuses on backend for an E-Commerce Website. This application is using the Node.js, Express.js and Sequelize.

## Installation

Install the required modules first.<br>
Run the Schema.<br>
Run the seeds.<br>
Npm start<br>
